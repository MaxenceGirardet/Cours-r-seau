Étiquettes

ARP (Address Resolution Protocol) :

  • Résout les adresses IPv4 en adresses MAC
  • Principe : Broadcast

ARP spoofing :

  • Répondre à une requête ARP dont on n'est pas le destinataire

Cache ARP :

  • Retient un certain temps les adresse MAC

RARP (Reverse ARP) → Obsolète :

  • Résout les adresses MAC en adresses IPv4

Broadcast :

  • Transmission de données à l'ensemble des machines d'un réseau (dans certains cas, c'est pour voir qui répond)

Multicast :

  • Diffusion d'un émetteur (source unique) vers un groupe de récepteurs

DHCP (Dynamic Host Configuration Protocol) :

  • Distribue des informations nécessaires à l’arrivée d’une machine sur le réseau
    • Adresse IP
    • Passerelle
  • Fonctionnement sous UDP
    • serveur port 67
    • client port 68
  • Fonctionnement Broadcast avec acquittement
    • Distribution d’adresse IP sur une intervalle d’adresse disponible
    • On peut choisir en fonction de l’adresse MAC afin de fixer les adresses IP
    • Client broadcast serveurs → serveurs répondent → client choisit et envoi à 1 serveur → serveur lui donne confirmation ou ack → client peut commencer à utiliser IP

Nom de domaine :

DNS (Domain Name System) :

  • Système de nommage hiérarchique qui transforme des noms en adresses IPv4 ou IPv6
  • Chaine alphanumérique utilisée dans le DNS → www.iut-clermont.fr.
  • TLD (Top Level Domain) :
    • Code de pays ou code générique → .fr , .museum
  • Organisation sous forme d’arbre
    • “ . ” est la racine unique
    • TLD
    • Domaine
    • FQDN (Fully Qualified Domain Name)

Résolution de nom :

  • Transforme un nom en IP
    • Static → hosts
    • Local → NIS , LDAP , DNS
    • Global → DNS

ICMP (Internet Control Message Protocol) :

  • Vérification des liens internet
  • PING , TRACEROUTE
  • Pas utilisé pour transmettre données d'app (fichiers ou des pages web), mais gérer et surveiller bon fonctionnement des com réseau

IP :

  • Adresse logique
    • Communication de machine en machine
  • Détection d’erreurs
  • Fragmentation
  • Acheminement

  • IPv4 → 32 bits
  • IPv6 → 128 bits (héxadécimal)

  • Chaque machine possède sa propre adresse logique
  • IPv4
    • Classe A → de 0.0.0.0 à 127.255.255.255 /8
    • Classe B → de 128.0.0.0 à 191.255.255.255 /16
    • Classe C → de 192.0.0.0 à 223.255.255.255 /24

  • IPv4 réseau privé :
    • 10.0.0.0 /8 jusqu’à 10.255.255.255
    • 172.16.0.0 /12 jusqu’à 172.31.255.255
    • 192.168.0.0 /16 → jusqu’à 192.168.255.255
